get_jpeg_V1


Example of how to run
    python "get_jpeg_V1.py" --user admin --Filename New_image --SequNo 1
    
Arguments 
    --user _______
        Camera username used to login
        If you do not include this then you will be asked to enter it when the application runs
    
    --pass _______
        Camera password used to login.
        If you do not include this then you will be asked to enter it when the application runs
    
    --Filename 
        Filename for the images excluding the file extention
        
    --SequNo
        Sequence number to be appended at the front of the filename. This is used to override the default of 0
